Getting Started
You can follow this tutorial on your own computer or on this website. In order to program with python on your computer, it must be installed. On macOS, Python 2 comes pre-installed. You can download and install python from python.org. Python files are saved using a .py extension. However, python can be coded straight from the terminal as well. On Mac, open Terminal. On Windows, open Command Prompt. Next, type "python" and press enter. From here on, you can program with python right in the terminal! To exit python, type exit(). 
>>> print("Hello World!")
Hello World!

Output
In programming, the output is what is displayed to the user when running a program. In python, the print() function is used to display text. This is one of the many built-in functions that come with Python. Like other functions in Python, the name of the function is followed by parentheses and takes an argument, or multiple arguments. An argument is what is put inside the parentheses. The print statement outputs whatever is put inside the parentheses. Python 2 does not require using parentheses when using print, but Python 3 does. To display numbers, the numbers can go straight in the parentheses. To print text, it must be surrounded with quotes. This is known as a string. Other data types, such as booleans, can also be printed. 
>>> print(100)
100
>>> print("Print this.")
Print this.

Variables and Data Types
A variables is a storage space that holds a specific value. In some programming languages, such as Java, variables can be initalized without a value. In Python, this is not the case. A variable in Python must be assigned some sort of value when it is initalized. A variable assignment statement is used to assign some sort of value to a variable. This value can be of multiple different data types. These data types are: numbers, strings, booleans, lists, dictionaries, and tuples.

There are four kinds of numeric data types: integers, long integers, floating point numbers, and complex numbers. An integer is just a "normal" number, positive or negative. A long integer is larger than an integer. An integer holds up to 32 bits of data, and anything larger becomes a long. A float is a number with a decimal. These three data types are used more often than complex numbers.

A boolean is a data type with two possible values: True and False. A string can be made up of any characters (numbers, letters, and symbols) and is generally surrounded by double quotes "", but can also be surrounded by single quotes ''.

Here are some examples of the most common data types:

Integer Float String Boolean List Tuple
15845098374268340 523987.957 "abcABC123" False [1, 542, "abc", "s"] (2, 43, 63, 3)

Since Python variables cannot be initialized or declared without a value assignment, a variable's data type does not need to be explicitly stated. A variable is assigned a value by using an equal sign "=" to the right of the variable name. The equal sign is followed by the value you want to assign the variable. 
>>> varInt = 4  #int
>>> varStr = "hello"  #string
>>> varBool = True  #boolean
>>> varList = [2, 12 , 5, "abc"]  #list
>>> varTuple = (1, 34, 5, 9) #tuple
(Note: In Python, the number sign "#" indicates a single line comment. This means that whatever characters come after the "#" on the same line are not read by the python interpreter. Comments are made to help whoever is reading the code to understand it. A multi-line comment is surrounded by three single quotes.) 
>>> #This is a single line comment.

When naming variables, there are a few rules to keep in mind:
    1) It cannot start with a number. It must start with a letter or some other acceptable ASCII character.
    2) A variable name can be made up of any ASCII characters, not including Python operators or characters with other specific functions (eg. "+", "-", "*", "-", "#", "%", ",", ".", "{}", "()", and many others.)
    3) When naming a variable made up of multiple words, you can use camelCase or underscores to seperate the words since spaces are not allowed. This is not a rule, but it makes reading the code much easier.
    4) When assigning a constant (a variable set to a permanent value), it is common practice to use all capital letters. This is also not a rule.
A variable can be displayed to the user of a program by using the print statement. The variable name goes inside the parentheses . 
>>> number = 974
>>> print(number)
974

Strings
A string is a sequence of characters surrounded by either single or double quotes. 
>>> name = "bob"
>>> name2 = 'bob'
>>> print(name)
bob
>>> print(name2)
bob

An advantage of using double quotes is that you can use single quotes inside the string. If a string is defined using single quotes, then you cannot use a single quote within the string. Otherwise, the string will break. 
>>> sentence = "Hello, I'm Bob!"
>>> print(sentence)
Hello, I'm Bob!

Remember, you can also print a string by directly entering the string as an argument in the print function rather than using a variable. 
>>> print("This is a string.")
This is a string.
>>>print("Hi Prarthana")
Error 403 -Can't find Prarthana :/

Getting Input
Getting input from the user is very important for some programs. For example, if you want to ask the user for two numbers to add, you need a way to get the input and store it. In python 2, the raw_input() function is used. In python 3, the input() function is used. Both of these functions do the same thing, which is they return whatever the user types in as a string. In python 2, input() also takes user input, but not in the same way. The difference is that it evaluates the characters inputed with python. This means that if a number is entered, input() will return the input with a numerical data type (such as aninteger or a float). If a boolean value (True or False) is entered, then it will return a boolean value. If the input is a string or a set of random characters, it must be surrounded by quotes. Otherwise, it will return an error. With raw_input(), all input is regarded as a string.

Both the raw_input function of python 2 and the input() function of python 3 work the same way. Inside the parentheses, you can put a string of characters that is displayed to the user when the program asks for input. If you leave the parentheses empty, no text will be displayed to the user. Generally, instructions or a question are put into the parentheses so the user knows what the program is asking for.

Python 2: 
>>> raw_input("Enter anything: ")
Enter a word: random words
'random words'

Python 3: 
>>> input("Enter anything: ")
Enter a word: random words
'random words'

In order to use the input, it must be stored. To do this, you can assign a variable to the input. The variable will hold the string value of the input. If you do not want to store the input as a string, you can convert it in the same statement. For example, if you want to store the input as an integer value, you can use the int() function. 

>>> var = input("Enter some text: ")
Enter some text: this is some text
>>> var2 = int(input("Enter a number: "))
Enter a number: 52
>>>print(var)
this is some text
>>> print(var2 + 3)
55

Operators
Math is a large part of programming and is used from the simplest of calculating to more complicated computing. In order to use math with python, or any other programming language, you need to learn the operators. These operators include "+", "-" ,"x", "/", and more. You likely have already seen these four basic operators in any math class. However, there are more that you probably have not seen before, such as mod ("%"). Mod is an operation that finds the remainder when a number is divided by another number. This can be useful in determining if a number is odd, even, or a factor of some number. These are the five most basic and important mathematical operators in python. 

>>> num1 = 5
>>> num2 = 4
>>> sum = num1 + num2
>>> diff = num1 - num2
>>> product = num1 * num2
>>> quotient = num1 / num2
>>> mod = num1 % num2
>>> print(sum)
9
>>> print(diff)
1
>>> print(product)
20
>>> print(quotient)
1.25
>>> print(mod)
1
